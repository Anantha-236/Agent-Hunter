# Contributing Policy

This repository is owner-controlled.

## Authorization Required

Any code change, feature addition, bug fix, documentation update, or distribution requires explicit written approval from the repository owner before implementation.

## Pull Requests

- Open an issue and request approval first.
- Submit a pull request only after approval.
- Owner approval is mandatory for merge.

## Prohibited Without Approval

- Unapproved forks for development use
- Redistributing modified versions
- Commercial or personal derivative use
- Publishing mirrors with altered code

## Legal Notice

Unauthorized use, copying, modification, redistribution, or derivative work may result in legal action.
